# Titanic Veri Analizi Projesi

Bu projede Titanic gemisinde bulunan yolcuların verileri analiz edilmiştir.

## Kullanılan Teknolojiler
- Python
- Pandas
- Matplotlib
- Seaborn

## Proje İçeriği
- Veri seti tanıtımı
- Tanımsal istatistikler
- Veri görselleştirme (5 farklı grafik)
- Bulgular ve yorumlar

## Veri Seti
Titanic yolcu veri seti kullanılmıştır. Veri seti yolcuların yaş, cinsiyet, bilet ücreti ve hayatta kalma durumlarını içermektedir.

## Dosyalar
- titanic.ipynb → Analiz kodları
- titanic.html → Rapor çıktısı
- Titanic-Dataset.csv → Veri seti
